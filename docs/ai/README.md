# LectureOS AI 협업 문서

이 폴더는 LectureOS를 AI 코딩 에이전트와 함께 개발할 때 사용하는 운영 규칙과 프롬프트를 보관한다.

## 문서 목록

- `AI_COLLABORATION_WORKFLOW.md`: 사람, ChatGPT, Codex의 역할과 작업 흐름
- `CODEX_BLUEPRINT_REVIEW_PROMPT.md`: Blueprint 문서 검토용 Codex 표준 프롬프트
- `DOCUMENT_COMMIT_POLICY.md`: 문서 단위 커밋 정책
- `DOCUMENT_REVIEW_CHECKLIST.md`: AI 작성 문서 승인 전 체크리스트

## 기본 원칙

1. 제품 방향과 핵심 의사결정은 사람이 확정한다.
2. ChatGPT는 제품 문서의 기준 초안을 작성한다.
3. Codex는 저장소 조사, 문서 간 일관성 검토, 기술적 현실성 검토를 담당한다.
4. Codex가 제품 철학을 임의로 변경하지 못하게 한다.
5. 문서 변경은 검토 후 문서별로 커밋한다.
6. 구현은 Blueprint가 승인된 뒤 시작한다.
